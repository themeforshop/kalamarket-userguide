    Home Page Sections

    Below guide listing sections of KalaMarket #1 theme.
    Other layout is the same guide. If it is hard for you to use, please submit a ticket to the Support system. Thank you.

    KalaMarket is the Sidebar layout, it includes Left sidebar and Right content.

    We will guide the Right content first.

    Slideshow

    • If the section existed (used preset), you only need click to section to edit.

    • If the store don't have it, only need click to button and select Slideshow.

    General Settings: Display on, Show Full width, Style value (Margin,Z-Index), Slide effect (include 4 effects: Sliding, Fading, Cube, Flipping ), Auto-rotate slides, Slide width in px (Desktop), Slide height in px (Desktop), Enable Navigaiton, Navigation Position and Enable Pagination.

    The Important area is the Content area (bottom). It will listing all slide of Slideshow. Click to Slide to edit. If you want add more slide, click to

    For the Image slide, You will got options:

    • Slide Background: Upload image to show. Also can set Slide link, Captions Entrances Effect and Captions Exits Effect

    • Title: It's heading of the caption.

    • Description: it's description of the caption.

    • Action: it't action button of the caption.

    • Other captions (maximun is 6) you can set below.

    With each caption element, you need set the style and position to make it show the like your idea. These options area Style, horizontal position(x), align with horizontal position, vertical position(y), layer order(z)


    For the Video slide, You will got options:

    • Video link: input the embed video.

    • Auto play.

    • Image loader.

    Information

    • If the section existed (used preset), you only need click to Information section to edit. The name will depend on Title text. So, you need temporary remove the title to have the right section name.

    • If the store don't have it, only need click to button and select Information.

    Collection settings: Display on and background color.

    Content settings: listong all information elements (Icon, heading, description and Responsive column).



    Banner Full

    • If the section existed (used preset), you only need click to Banner Full section to edit.

    • If the store don't have it, only need click to button and select Banner Full.

    Collection settings: Display on, upload banner background, banner caption and action button.



    Collection Grid

    • If the section existed (used preset), you only need click to Collection Grid section to edit. The name will depend on Title text. So, you need temporary remove the title to have the right section name.

    • If the store don't have it, only need click to button and select Collection Grid.

    Layout settings: Display on, Full width option.

    Section settings: Title, Sub Title, Collection to show products, limit show, image size and responsive columns. Banner settings.



    Banner Grid

    • If the section existed (used preset), you only need click to Banner Grid section to edit.

    • If the store don't have it, only need click to button and select Banner Grid.

    General Settings: Display on.

    Block conent: listing all banners: upload image, set link to and Responsive settings.



    Left Sidebar

    • If the section existed (used preset), you only need click to Left Sidebar section to edit. The name will depend on Title text. So, you need temporary remove the title to have the right section name.

    Section settings: Display on, Category settings.

    There is 9 block type: You can add more by click to



    Icon Category

    Use to show icon for each Category



    Sub Category Setting

    Use to show Name of Category



    Sub Category Content

    Use to show dropdown content of Category



    Banner

    Options listing: upload image, action text and action link.



    Collection

    Options listing: Title, collection to show products and limit show.



    Blog

    Options listing: Title, blog to show articles and limit show.



    Review

    Options listing: Background color, review elements (active, upload avatar, review text, name and job)

    Brand

    Options listing: brand elements (upload image and link to)

    Instagram

    Options listing: Insgram photos elements (upload image and link to)
